Detailed Description

Maze generation for simulation This module contains functions for creating a maze object. The maze path is generated as a tree, and the function returns a list of edges between the nodes of the grid. 

Create a 2D maze

get the edge coordinates, given a box in grid Given the x and y parameters of the box, return the corresponding wall in the maze, represented as a list of edges. The return object is represented as a 3D array such that: [[(x, y of node 1), (x, y of node 2)], [(x, y of node 2), (x, y of node 1)] (both orders are listed to prevent any orientation mismatch, also see f_print).
